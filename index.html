HTML

  • HTML stands for Hyper Text Markup Language
  • HTML is in charge of creating the "skeleton" of websites
  • HTML is one of the main languages of the web

CSS

  • CSS adds style to HTML elements
  • CSS uses "Class" and "id" selectors to target HTML elements and sections
  • CSS stands for Cascading Style Sheets

Git

  • Git is both a language and an application used by developers
  • Git was created by the same creator of Linux
  • Git is used to execute commands in terminal on Mac and Gitbash on Windows

JavaScript

  • JavaScript is used in conjunction with HTML and CSS
  • JavaScript provides logic and interaction within websites
  • JavaScript is considered a true programming language, HTML and CSS are not
